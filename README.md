## ✨ Features
- Choose from popular meme templates
- Real-time search
- Modern UI with smooth animations
- Built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**

## 🛠️ Tech Stack
- Frontend: React + Vite
- Styling: Tailwind CSS
- Animations: Framer Motion
- Icons: Lucide React
